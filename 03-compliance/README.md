## 📝 Templates & Checklists

### FADP Compliance Checklist

```markdown
□ Data inventory documented
□ Legal basis for each processing activity
□ Privacy policy updated for FADP
□ Data subject rights procedures
□ Cross-border transfer assessment
□ Processor agreements (DPA) in place
□ Security measures documented
□ Breach notification process
□ Privacy impact assessment (if required)
□ DPO appointed (if required)
```

### AI-Specific Checklist

```markdown
□ AI model documentation
□ Training data provenance
□ Bias assessment completed
□ Explainability measures
□ Human oversight for automated decisions
□ Model monitoring in place
□ Incident response for AI failures
□ Regular model revalidation
```
