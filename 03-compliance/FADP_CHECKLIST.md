# FADP Compliance Checklist for AI Systems
# Swiss Federal Act on Data Protection (nDSG)
# Effective: September 1, 2023

## 1. Data Inventory & Mapping

- [ ] All personal data processing activities documented
- [ ] Data flow diagrams created
- [ ] Data categories identified (sensitive vs. non-sensitive)
- [ ] Data retention periods defined
- [ ] Third-party data sharing documented

## 2. Legal Basis (Art. 6 FADP)

- [ ] Legal basis identified for each processing activity
  - [ ] Consent
  - [ ] Contract performance
  - [ ] Legal obligation
  - [ ] Vital interests
  - [ ] Public interest
  - [ ] Legitimate interests
- [ ] Consent mechanisms implemented (if applicable)
- [ ] Consent records maintained

## 3. Privacy by Design (Art. 7 FADP)

- [ ] Privacy considered at design phase
- [ ] Data minimization implemented
- [ ] Purpose limitation enforced
- [ ] Default privacy settings applied
- [ ] Anonymization/pseudonymization where possible

## 4. Data Security (Art. 8 FADP)

- [ ] Encryption at rest (AES-256)
- [ ] Encryption in transit (TLS 1.3)
- [ ] Access controls implemented
- [ ] Authentication mechanisms (MFA)
- [ ] Audit logging enabled
- [ ] Regular security assessments
- [ ] Incident response plan

## 5. Data Subject Rights

- [ ] Right of access (Art. 25)
- [ ] Right to rectification (Art. 32)
- [ ] Right to erasure (Art. 32)
- [ ] Right to data portability (Art. 28)
- [ ] Right to object (Art. 30)
- [ ] Automated decision-making disclosure (Art. 21)

## 6. Cross-Border Transfers (Art. 16-17 FADP)

- [ ] Data residency requirements assessed
- [ ] Adequate country list checked
- [ ] Standard contractual clauses (if needed)
- [ ] Binding corporate rules (if applicable)
- [ ] Transfer impact assessment

## 7. AI-Specific Requirements

- [ ] AI model documentation
- [ ] Training data provenance
- [ ] Bias assessment completed
- [ ] Explainability measures implemented
- [ ] Human oversight for automated decisions
- [ ] Model monitoring in place
- [ ] Regular model revalidation

## 8. Documentation

- [ ] Privacy policy updated
- [ ] Data processing agreements (DPA)
- [ ] Records of processing activities
- [ ] Privacy impact assessment (PIA)
- [ ] Data breach notification procedures

## 9. Organization

- [ ] Data Protection Officer (if required)
- [ ] Staff training completed
- [ ] Vendor management process
- [ ] Regular compliance audits
